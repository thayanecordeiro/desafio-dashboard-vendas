# Dashboard de Vendas

Projeto criado para o desafio da DIO.

## Dados utilizados
Base `base.xlsx` fornecida pela DIO, contendo informações de assinaturas, preços e datas.

## Descrição do projeto
- Criei 3 tabelas dinâmicas:
  - Vendas por Produto
  - Vendas por Mês (agrupado por mês)
  - Vendas por Plano
- Criei gráficos para facilitar a visualização (opcional).

## Como reproduzir
1. Abra o arquivo `dashboard_vendas.xlsx` no Excel.
2. Navegue pelas abas para ver as tabelas dinâmicas.
3. Atualize os dados se desejar clicando em qualquer tabela dinâmica e usando o botão “Atualizar”.
